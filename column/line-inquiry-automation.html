LINE公式の問い合わせ対応を自動化する方法
——キーワード返信からGAS連携まで

2026年5月20日

「LINEの問い合わせ対応に時間を取られている」「夜中にメッセージが来て翌朝まで返せない」——こうした悩みは、個人事業主やスモールビジネスのオーナーによく聞きます。LINE公式アカウントには問い合わせ対応を自動化する機能が備わっており、設定次第で手動対応の大部分を減らすことができます。

シンプルなキーワード自動返信から、GASを使った高度な連携まで、段階別に解説します。

① キーワード自動返信の基本設定

LINE公式の管理画面には「応答メッセージ」という機能があります。特定のキーワードをユーザーが送信したとき、あらかじめ設定したメッセージを自動で返信する仕組みです。追加費用なし、プログラミング不要で設定できます。

設定手順
  1. LINE公式アカウントマネージャーにログイン
  2. 「応答」→「応答メッセージ」を選択
  3. 「作成」→ キーワードと返信内容を入力
  4. 「保存」で即日有効化

1つのキーワードに対して最大5パターンの返信を設定でき、ランダムで返信する設定も可能です。

② よくある質問をキーワードで自動化する

どのビジネスにも「よく聞かれること」があります。それをキーワードに設定しておくだけで、同じ質問に何度も手動で返す手間がなくなります。

キーワード例 返信内容の例
料金 / 値段 / いくら 料金表ページのURL＋「ご不明点はお気軽にどうぞ」
予約 / 申込み 予約フォームへのリンク
場所 / アクセス 住所・地図リンク・最寄り駅案内
営業時間 / 何時 営業時間と定休日の案内
キャンセル / 変更 キャンセルポリシーと連絡先

「完全一致」だけでなく「部分一致」にも対応しているため、「予約したい」「予約できますか」など表現が異なるメッセージにも反応させられます。

③ Googleフォーム連携で情報収集を自動化する

問い合わせの初動を自動返信でカバーしたら、次は「情報収集」の自動化です。キーワード返信のメッセージにGoogleフォームへのリンクを入れておくことで、ユーザーが自分で必要事項を記入→管理者に通知が届く流れを作れます。

Googleフォームは無料で作成でき、回答はスプレッドシートに自動で蓄積されます。「問い合わせが来るたびにメモする」という作業がなくなります。

フォームには名前・連絡先・問い合わせ内容の他、「希望の対応方法（LINE / 電話 / メール）」を入れておくと、その後のやり取りがスムーズになります。

④ GASを使った高度な自動返信

キーワード自動返信だけでは対応しきれない複雑な要件には、GAS（Google Apps Script）との連携が有効です。GASを使うと、LINE公式の標準機能では実現できない自動化が可能になります。

GAS連携でできること（例）
  • フォームに回答があったタイミングで、自動でLINEに確認メッセージを送信
  • スプレッドシートに登録された顧客情報をもとに、個別メッセージを自動送信
  • 予約日時の前日に自動でリマインドメッセージを送信
  • 問い合わせ内容に応じてメッセージの内容を分岐させる
GASはGoogleが提供するプログラミング環境です。設定には技術的な知識が必要なため、自動化の範囲が広い場合は専門家への依頼も選択肢になります。

まとめ——自動化のステップ

問い合わせ対応の自動化は、難易度の低いところから順番に整えていくのが効率的です。

  1. キーワード自動返信：管理画面から設定。追加費用なし、即日対応可能
  2. Googleフォーム連携：リンクを貼るだけ。情報収集と管理が自動化
  3. GAS連携：技術的なカスタマイズが必要。本格的な自動化に対応

まずステップ1のキーワード自動返信だけでも設定すると、夜間・休日の問い合わせへの即時対応が実現します。「全部まとめて自動化したい」という方は、構築を専門家に依頼するのが最短ルートです。
