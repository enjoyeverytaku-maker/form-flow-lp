Glossary

用語解説

LINE公式アカウントの活用に登場する用語をわかりやすく解説します

STEP DELIVERY

ステップ配信とは

友だち追加後、設定したスケジュールに沿って自動でメッセージを届ける機能。見込み客の育成に使われる。

詳しく読む →
RICH MENU

リッチメニューとは

LINEトーク画面の下部に固定表示されるメニュー。ボタンにURLや応答メッセージを設定できる。

詳しく読む →
GOOGLE APPS SCRIPT

GASとは

Googleが提供する無料のプログラム実行環境。LINEと連携することで自動返信・顧客記録・予約管理などを月額0円で実現できる。

詳しく読む →
WEBHOOK

Webhookとは

「何か起きたとき、外部に通知する」仕組み。LINEではメッセージが届いたときにGASへ自動で通知を送るために使う。

詳しく読む →
LINE OFFICIAL ACCOUNT

LINE公式アカウントとは

企業・店舗・個人事業主が顧客とLINEでやり取りするための専用アカウント。個人LINEとの違いや無料でできることを解説。

詳しく読む →
KEYWORD REPLY

キーワード自動返信とは

「予約」「料金」などの特定キーワードを受け取ったとき、あらかじめ設定した返信を自動で届ける機能。問い合わせ対応の手間を大幅に削減できる。

詳しく読む →
BROADCAST

一斉配信とは

友だち全員または絞り込んだグループに同時にメッセージを届ける機能。フリープランで月200通まで無料。配信タイミングのコツも解説。
