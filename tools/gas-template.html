自動返信 / Webhook

キーワード自動返信 + 顧客記録テンプレート

LINEに届いたメッセージのキーワードを判定して自動返信し、送信者の情報をスプレッドシートに記録します。最もよく使われる基本テンプレートです。

  • キーワード判定（部分一致対応）
  • 複数キーワード・複数返信文の設定
  • ユーザーIDと受信日時をスプレッドシートに自動記録
  • Webhook受信エラー時のログ出力

フォーム→カレンダー予約 + LINE確認送信テンプレート

一斉送信 / バッチ処理

友だちリスト一斉送信テンプレート

スプレッドシートに登録した友だちリストへ一斉送信します。チャンク処理で大人数にも対応。タイムアウト対策と送信ログ記録を実装済みです。

  • スプレッドシートの友だちリストをもとに送信
  • 50件ずつのチャンク処理でタイムアウト防止
  • 送信結果（成功・失敗）をログに記録
  • 送信日時の指定（タイマートリガー設定例付き）
